# Flutter Boost 集成 Flutter Module 示例
这是一个演示如何使用 Flutter Boost 将 Flutter 模块集成到 Android 项目中的示例。Flutter Boost 是一个框架，它允许你将 Flutter 嵌入到现有的 Android 应用中。
## 项目简介
本示例项目展示了如何将一个 Flutter 模块与一个 Android 项目相结合，以便在 Android 应用中使用 Flutter 功能。通过这个示例，你可以了解如何设置 Flutter 项目、配置 Flutter Boost、以及如何将 Flutter 模块集成到 Android 项目中。
## 安装说明
1. 克隆本项目仓库：
   ```sh
   https://github.com/chenfg1/AndroidBindFlutterSimply
   ```
2. 在 Android Studio 中打开项目。
## 使用说明
1. 构建项目并在模拟器或物理设备上运行。
2. 在主屏幕上点击“打开 Flutter 模块”按钮，打开 Flutter 模块。
3. Flutter 模块应该被显示出来。
